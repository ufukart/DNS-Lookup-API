🌐 DNS Lookup API

A comprehensive DNS lookup service with support for multiple record types and resolvers

📡 API Endpoints

GET/api/lookup

domain (required) - Domain name or IP address

resolver (optional) - Choose from: google, cloudflare, opendns, quad9, adguard, dnswatch, comodo, cleanbrowsing, ultradns

GET/health

Health check endpoint - Monitor service availability

GET/metrics

Prometheus metrics endpoint - Performance and usage statistics

💡 Example Usage

🔍 Domain lookup: /api/lookup?domain=example.com&resolver=cloudflare
🔄 PTR (reverse DNS) lookup: /api/lookup?domain=8.8.8.8&resolver=google
🎯 Simple query (default resolver): /api/lookup?domain=github.com

📋 Supported Record Types

A AAAA NS MX TXT CNAME SOA SRV CAA PTR
